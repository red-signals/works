Vmware中设置固定ip

发表于2024-04-29|更新于2024-06-03|网络本地服务
|字数总计:1.2k|阅读时长:4分钟|阅读量:

VMware中的虚拟机每次重启IP都会变化，为了解决这个问题需要设置静态IP。

一、查看虚拟网址信息

在VMware安装的宿主机查看网址信息，发现多出如下两个虚拟网卡

  1. 以太网适配器 VMware Network Adapter VMnet1
  2. 以太网适配器 VMware Network Adapter VMnet8

image.png

二、将虚拟机中的系统静态IP绑定到VMnet8上

获取******VMnet8******虚拟网卡的网段

由于IP一直变化，我们想固定IP，可以通过将虚拟机中的系统静态IP绑定到VMnet8上来实现。

通过ipconfig -all查询到VMnet8网卡的网段为192.168.211

，这个等会需要使用。

三、虚拟机中网络配置

进入安装好的CentOS7中配置网络IP，先做这两个操作 ->

*3.1* *查看网卡信息*

*3.2* *编辑网卡信息配置*

查看配置文件

cd etc/sysconfig/network-scripts/

编辑配置文件

vi etc/sysconfig/network-scripts/ifcfg-eno16777736

配置文件的初始信息如下

image.png

修改后的配置文件信息如下 ->

IPADDR=”192.168.211.104” # 当前设置的固定IP地址，注意网段与VMnet8一致

NETMASK=”255.255.255.0” # 子网掩码，VMnet8的子网掩码，上面也可以看到

DNS1=”192.168.3.1” # 宿主机的DNS，可以通过ipconfig查看

GATEWAY=”192.168.211.2” # 网关地址，前面第二步中网关地址

image.png

*3.3* *重启网络*

service network restart

image.png

*3.4* ***查看虚拟机网络****IP*

此时已经为我们修改后的固定IP 192.168.211.104

image.png

*3.5* ***重启虚拟机再次查看****IP*

仍然是192.168.211.104
